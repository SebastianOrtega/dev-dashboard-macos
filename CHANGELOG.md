# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-04-13

- Initial public release of Dev Dashboard for macOS.
- Added local React + Express dashboard for active development processes and ports.
- Added native Swift menubar app with search, warnings, process actions, and project termination.
- Added local packaging, signing, and notarization workflow for the macOS app.
- Added prebuilt notarized `.app` and `.zip` release assets for direct download.
